# Imagegen Figure Candidates

These PNG figures were generated with the built-in `$imagegen` workflow as
bitmap visual candidates for the patch-verification paper.

They are intended for graphical abstracts, presentations, or visual drafts.
For exact experimental values, prefer the reproducible Matplotlib vector
figures in `docs/figures/`.

## Files

- `imagegen_framework.png`: high-level patch-verification workflow.
- `imagegen_evidence_boundary.png`: evidence visibility by review condition.
- `imagegen_tradeoff.png`: conceptual safety/recall tradeoff summary.
- `imagegen_claim_boundary.png`: supported claim-boundary summary.

## Generation Notes

- Mode: built-in `image_gen` tool.
- Date: 2026-06-06.
- Source originals remain under the Codex generated-images directory.
- Project copies are stored here so paper assets do not depend on Codex's
  default generated-image location.
- These are raster images and should not replace exact vector charts unless the
  target venue accepts conceptual graphical abstracts.
